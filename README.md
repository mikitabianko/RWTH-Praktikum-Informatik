# RWTH Praktikum Informatik

C++-Lösungen und Übungsprojekte aus den Informatikpraktika an der RWTH Aachen.

## Projekte

### ET-Praktikum 1

Der Ordner [`ET-Praktikum1`](ET-Praktikum1) enthält neun Versuche zur
C++-Programmierung:

- Grundlagen, Funktionen und Felder
- Tests und Dokumentation mit Doxygen
- Klassen und dynamische Datenstrukturen
- STL, Operatorüberladung, Vererbung und Polymorphie
- GUI-Programmierung und Wegsuche mit Qt

Die Versuche 1–8 sind Eclipse-CDT-Projekte. Versuch 9 wird mit Qt Creator über
`ET-Praktikum1/Versuch09/Versuch09.pro` geöffnet.

### CE-Praktikum

Der Ordner [`CE-Praktikum`](CE-Praktikum) enthält eine objektorientierte
Verkehrssimulation in zwei Aufgabenblöcken. Behandelt werden unter anderem
Smart Pointer, Vererbung, Templates, Exceptions und eine grafische Simulation.

Eine ausführliche Aufgabenübersicht befindet sich im
[`CE-Praktikum/readme.md`](CE-Praktikum/readme.md).

## Wichtiger Hinweis

Dieses Repository ist ausschließlich als **Lernhilfe und Orientierung für die
eigene Vorbereitung** gedacht. Der enthaltene Code sollte **nicht unverändert
oder als eigene Leistung zur Abgabe eingereicht** werden. Aufgabenstellungen,
Vorlagen und Bewertungskriterien können sich außerdem jederzeit ändern.

Bearbeiten Sie die Aufgaben selbstständig, gleichen Sie Ihre Lösung mit der
aktuell ausgegebenen Aufgabenstellung ab und stellen Sie sicher, dass Sie jeden
Teil des Codes verstehen und erklären können. Der Code kann Fehler oder
unvollständige Lösungen enthalten. Eine vollständige Übereinstimmung mit
zukünftigen Durchgängen des Praktikums und deren Anforderungen kann nicht
garantiert werden. Für die Richtigkeit, Vollständigkeit und die aus der
Verwendung der Inhalte entstehenden Folgen wird keine Gewähr übernommen.
